# EMS-Sorter
EMS Sorter – это утилита для автоматического перемещения скриншотов из папки-источника (например, папка NVIDIA Overlay) в папки назначения, соответствующие вашим рабочим сменам на сервере MAJESTIC RP (оптимально для сервера Denver). Программа отслеживает появление новых скриншотов, переносит их в нужную категорию и автоматически ведёт статистику заработанных баллов

# Назначение
Помогает систематизировать скриншоты по видам деятельности (ПМП, таблетки, вакцины, мед. справки и т.д.)

Считает баллы на основе реального количества файлов в каждой папке – никаких ручных правок статистики не требуется

Исключает ошибки при подсчёте: баллы всегда равны количество скриншотов × вес категории

Ускоряет работу сотрудников EMS (медицинской службы) на сервере

# Структура
EMS-Sorter.ps1 – основной скрипт

EMS.bat – запускаемый файл (содержит ту же версию для проверки целостности)

config.ini – файл настроек (создаётся при первом запуске). В нём хранятся пути к папке-источнику и ко всем папкам назначения

session.log – журнал всех операций (создаётся автоматически)

stats.ini - файл, куда сохроняются ваши таблетки, ПМП и другие работы

# Как это работает
Вы указываете (при входе в программу) в config.ini путь к папке, куда игра/программа сохраняет скриншоты (источник), и пути для каждой категории (назначения)

Запускаете EMS.bat и выбираете нужный режим (например, ПМП день город)

Программа начинает следить за появлением новых файлов в папке-источнике

Как только появляется скриншот (форматы .png, .jpg, .jpeg), он копируется в выбранную папку назначения, а исходник удаляется (перемещается)

Баллы пересчитываются моментально – в главном меню и в статистике всегда отображаются актуальные значения, основанные на текущем содержимом папок

# Меню и управление
В главном меню отображаются:

Общий счёт – общее число скриншотов и суммарное количество баллов

Список режимов (1–12) – выбор категории для отслеживания

Настройки путей (13) – изменение путей без ручного редактирования config.ini 

Показать настройки (14) – просмотр текущих путей

Статистика (15) – детальный отчёт по каждой категории

Очистить баллы (БЕЗ УДАЛЕНИЯ В ПАПКАХ) - R1 (r1)

Переместить скриншоты и очистить баллы - R2 (r2)

Выход (0) – закрытие программы.

В режиме отслеживания:

Нажмите B или русскую "И" для выхода в главное меню. Будет задан вопрос Выйти в главное меню? (Y/N) – ответьте Y (выйти) или N (остаться)

Ctrl+C также работает, но не рекомендуется, так как cmd выдает стандартное сообщение Terminate batch job. Лучше использовать B.
