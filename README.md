# EMS-Sorter
EMS Sorter – это утилита для автоматического перемещения скриншотов из папки-источника (например, папка NVIDIA Overlay) в папки назначения, соответствующие вашим рабочим сменам на сервере MAJESTIC RP. Программа отслеживает появление новых скриншотов, переносит их в нужную категорию и автоматически ведёт статистику заработанных баллов.
